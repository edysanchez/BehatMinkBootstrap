BehatMinkBootstrap
==================

Repo ready to start with [Behat](http://github.com/Behat/Behat) and [Mink Extension](http://github.com/Behat/MinkExtension)

##Installing:

```bash
$>curl -sS https://getcomposer.org/installer | php
$>php composer.phar install
```

##Running:

```bash
$>bin/behat 
```
